##安装

1)下载完毕后，将<code>laravel51.sql</code>导入数据库（默认名为laravel51）。

2)将.env.example重命名为.env

内容如下：
DB_HOST=localhost
DB_DATABASE=laravel51
DB_USERNAME=root
DB_PASSWORD=


3) 本地环境切换到文件夹下运行命令 <code>php artisan serve</code> （打开<code>localhost:8000</code>即可看到效果）
